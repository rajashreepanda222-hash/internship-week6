# Interactive Sales Dashboard

## Project Overview

This project focuses on analyzing sales data using Python visualization libraries such as Seaborn, Matplotlib, and Plotly. The dashboard provides insights into sales trends, product performance, customer behavior, and regional sales distribution through both static and interactive visualizations.

## Objectives

* Analyze sales performance across different products and regions.
* Visualize sales trends over time.
* Understand sales distribution using statistical plots.
* Explore relationships between numerical variables.
* Build an interactive dashboard for better business decision-making.

## Dataset Information

The dataset contains the following attributes:

| Column      | Description                |
| ----------- | -------------------------- |
| Date        | Transaction date           |
| Product     | Product name               |
| Quantity    | Number of units sold       |
| Price       | Unit price                 |
| Customer_ID | Unique customer identifier |
| Region      | Sales region               |
| Total_Sales | Quantity × Price           |

## Technologies Used

* Python
* Pandas

* Matplotlib

* Plotly

## Visualizations Created

### 1. Monthly Sales Trend

Line chart showing sales growth and decline over time.

### 2. Product Performance Analysis

Bar chart comparing total sales generated by each product.

### 3. Box Plot

Visualizes sales distribution and identifies outliers.

### 4. Violin Plot

Displays distribution density of sales across products.

### 5. Correlation Heatmap

Shows relationships among Quantity, Price, and Total Sales.

### 6. Regional Sales Analysis

Compares sales performance across regions.

### 7. Customer Sales Distribution

Histogram showing customer purchasing behavior.

### 8. Interactive Dashboard

Built using Plotly with multiple charts and interactive features.
